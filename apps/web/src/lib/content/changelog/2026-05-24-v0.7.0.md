Los avistamientos ahora sugieren automáticamente mascotas perdidas con perfil similar. Al abrir un avistamiento abierto aparece una tarjeta "¿Podría ser una de estas?" con las mejores coincidencias, indicadores de qué atributos coinciden y acceso directo al WhatsApp del dueño.

## Algoritmo de emparejamiento

- Cada avistamiento se compara contra todas las mascotas activas usando cinco señales: especie (40 pts), colonia (30 pts), color (10 pts), tamaño (10 pts) y recencia (10 pts)
- Los colores se comparan por **grupos de sinónimos en español** con normalización de acentos — "café" y "marrón" se reconocen como el mismo color, "negro con manchas blancas" coincide con "blanco y negro"
- El resultado se guarda en base de datos; la segunda visita es instantánea
- Umbral mínimo de 30 puntos para mostrarse; el anillo visual se vuelve ámbar al superar 60 puntos

## Componente MatchSuggestions

- **Anillo SVG de puntuación** a la derecha de cada tarjeta — el arco muestra el puntaje de 0 a 100 directamente
- **Pastillas de señales** — colonia, color y tamaño se iluminan en ámbar cuando coinciden con el avistamiento
- **CTA de contacto directo** — abre WhatsApp con un mensaje prellenado si el reporte tiene número de contacto

## API

- `POST /api/matches/[id]/verdict` — permite registrar si una sugerencia fue correcta o incorrecta para calibrar el algoritmo en el futuro

## Infraestructura

- Nueva tabla `match_results` con índice compuesto único `(spotted_pet_id, pet_id)`
- 11 pruebas unitarias con Vitest cubren `scoreMatch()` y `colorsOverlap()`
- Documentación del algoritmo en `packages/services/src/matching/README.md` para contribuciones de la comunidad
